# Markflow — Support

Help and support for **Markflow — Markdown Reader**, a native markdown reader and editor for iPhone.

## Frequently asked questions

### How do I open a file?

Tap **Browse** on the home screen, then pick any `.md` file from the Files app. You can also open a `.md` file from Mail, Safari downloads, or any other app's share sheet — pick Markflow as the destination.

### How do I edit a file?

Open a file, then tap **Edit** in the segmented picker at the top of the screen. A markdown toolbar appears above the keyboard for bold, italic, headings, lists, links, and code.

### How do I save my changes?

Tap the back arrow (top-left). You'll be asked to:

- **Save** — writes over the original file
- **Save as New File** — keeps the original untouched and saves a copy
- **Discard Changes** — throws away your edits

You can also tap the share button (top-right) to export an edited copy anywhere.

### Will Markflow change my original file?

Never — unless you explicitly choose **Save**. Everything else (Save as New File, Share) keeps your original untouched.

### What markdown flavors are supported?

GitHub-Flavored Markdown (GFM): headings, bold, italic, strikethrough, ordered/unordered lists, task lists, tables, code blocks with syntax highlighting, blockquotes, links, inline images, horizontal rules, and Mermaid diagrams.

### Why don't my images show up?

Markflow only supports **absolute image URLs** (e.g. `![](https://example.com/image.png)`). Relative paths like `![](./image.png)` don't resolve because the renderer is sandboxed.

### Does Markflow send any data anywhere?

No. Markflow makes zero network requests and collects zero data. No accounts, no analytics, no tracking, no ads. Your files stay on your device. See the [privacy policy](privacy-policy.md) for details.

### Can I use Markflow to write a new file from scratch?

Yes. Tap **Create** on the home screen for a blank starter document, or **Welcome to Markflow** for a tour document you can edit and save as your own.

### The app crashed or is behaving strangely
